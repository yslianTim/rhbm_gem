# User Documentation

This section is for users who want to build, install, and run RHBM-GEM without digging through repository-internal design notes.

## Recommended Reading Order

1. Read [`getting-started.md`](getting-started.md) for platform-specific prerequisites, build and install commands, Python verification steps, and troubleshooting.
2. Run [`../../examples/python/00_quickstart.py`](../../examples/python/00_quickstart.py) for a minimal Python workflow after installation.
3. Run [`../../examples/python/01_end_to_end_from_test_data.py`](../../examples/python/01_end_to_end_from_test_data.py) for an end-to-end example using test data.

If you need to modify project internals, extend commands, or understand repository architecture, switch to [`../developer/README.md`](../developer/README.md).
